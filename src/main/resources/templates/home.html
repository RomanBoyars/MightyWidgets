REST API Reference

Welcome to the Mighty Widgets REST API reference. You can use this API to work with widget-type objects in your application. Mighty widgets provides an API to create, delete, and get widget objects in the format of JSON. In the sections below, you can find documentation, expected HTTP response codes and sample requests to each available API method.
